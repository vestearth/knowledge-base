# Weekly Review

## Purpose

Use this once a week to keep the vault alive without turning it into architecture theater.

## How To Use

Use this once per week, not every time you edit a note.

Do not turn this into a long retrospective. The point is to move useful notes, prune stale ones, and keep the vault lightweight.

After finishing, update one note that clearly earned promotion or cleanup.

## Review Questions

- What did I capture this week?
- Which notes were actually used?
- Which decisions now deserve an ADR?
- Which prompts worked more than once?
- Which lessons should become reusable skills?
- Which open questions are still real?
- Which notes should be deleted or merged?

## Checklist

1. Review [[Inbox]].
2. Move useful items to project notes or knowledge units.
3. Apply [[Promotion Rule]].
4. Add or repair source links using [[Source Link Convention]].
5. Pick one note to improve.
6. Keep the review under 30 minutes.
