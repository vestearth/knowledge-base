# How To Use This Vault

## Start Here

Open [[Home]] first. It is the dashboard for the current seed project, active decisions, reusable notes, and the operating loop.

## Daily Flow

1. Capture loose ideas in [[Inbox]].
2. Keep project-specific understanding in project notes such as [[Project Map]] or [[Current Decisions]].
3. Turn repeated patterns into reusable notes like [[Verification Loop]] only after they have real reuse pressure.
4. Keep important claims tied to [[Source Link Convention]].

## Weekly Flow

Run [[Weekly Review]] once a week.

The goal is not to organize everything. The goal is to move only the notes that proved useful.

## Promotion Logic

Use [[Promotion Rule]] when deciding whether something should stay as a project note, become an ADR, move into the prompt library, or become a reusable skill.

## Guardrails

- Do not create structure before there is evidence.
- Do not promote notes because they look clean.
- Do not let summaries drift away from source files.
- Do not add a new repo map until a real reuse pattern appears.
