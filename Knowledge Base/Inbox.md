# Inbox

Use this note for fast capture when the idea is real but the final place is unclear.

## How To Use

Use this when you have a real insight, decision, prompt, lesson, or question but do not yet know its final home.

Do not use this for polished notes or final summaries.

After capturing an item here, process it during [[Weekly Review]] with [[Promotion Rule]].

## Capture Rule

Put something here when it is:

- a decision that is not ready for ADR
- a prompt that worked once but is not proven reusable
- a lesson from a task or bug
- an open question without a project home
- a source link worth revisiting
- a pattern that might become a skill

## Triage Rule

During weekly review, move each item to one of these places:

- project note
- reusable knowledge unit
- ADR
- prompt library
- open question
- delete if no longer useful
